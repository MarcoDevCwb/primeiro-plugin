# Product Vision - SubLoop

SubLoop is designed to become the rhythmic brain for basslines in underground electronic music.

## Core philosophy

- Fast results
- Musical groove first
- Style-driven creation
- Minimal interface, maximum outcome

## Target users

- Tech House producers
- Techno producers
- Minimal / Deep Tech artists
- Beginner to intermediate producers

## Long-term goal

Become a standard MIDI bassline tool integrated in modern DAW workflows.
